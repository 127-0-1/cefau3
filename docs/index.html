Cefau3 is an open source CEF wrapped library for AutoIt3, written in C and used syntax like C++. The best easy way to create your own web application.

Easy to Learn

Wrapped for AutoIt, so easy for learning.

Small Codebase

Cefau3's implementation is 25k sloc ANSI C without dependencies.

Object-oriented-like

Using AutoItObject, everything is object-like, or can use functionally for code.
